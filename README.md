# Brnde | Premium LinkedIn Brand Design

LinkedIn's #1 brand design agency. We help creators and founders build premium personal brands with strategic design.

## 🚀 Tech Stack

- **Framework**: React 19 (Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **SEO**: JSON-LD Structured Data, OpenGraph, and Twitter Cards

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/brnde-site.git
   ```
2. Navigate to the project directory:
   ```bash
   cd brnde-site
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the local development server:
```bash
npm run dev
```

### Building for Production
Create an optimized production build:
```bash
npm run build
```
